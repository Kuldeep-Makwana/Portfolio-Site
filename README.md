# 🌟 Adventure Portfolio

A creatively designed **personal portfolio** that gamifies your journey, achievements, skills, and past projects — making the user experience feel like an **adventure quest**.
This portfolio showcases your UI/UX sense, clean code, and ability to merge creativity with modern web development.

## 🚀 Features

### 🔱 **Interactive Adventure-Themed Navigation**

Navigate through sections like *Archer*, *Fighter*, *Quests*, *Achievements*, *Roles*, and more — each presenting skills or experiences using a story-driven style.

### 🎬 **Background Video & Music**

* Dynamic **background video** for immersive experience
* Optional **background music** for mood enhancement

### 🧩 **Projects (Quests) Display**

Dedicated quest pages showcasing:

* **NearBy BRTS Admin Panel**
* **DealHut E-Commerce Backend**
* **College Web Hackathon Project**
* **UI/UX Tasks & Design Projects**

Each quest includes screenshots, system walkthroughs, and project outcomes.

### 🗂️ **Well-Organized File Structure**

Assets such as images, backend screenshots, adventures, profile images, and UI/UX tasks are neatly arranged into folders for maintainability.

### 📝 **Custom Maintenance Page**

A dedicated **maintenance mode** page with animation + future-ready design.

### 📱 **Fully Responsive**

Designed to work smoothly across **mobile, tablet, and desktop** devices.

### 🧠 **Clean & Modern Codebase**

* Organized HTML structure
* Centralized CSS
* JavaScript separated into `scripts/main.js` for clarity
* Lightweight & fast performance

## 📂 Folder Structure

```
adventure-portfolio-main
├─ HTML Files (main pages)
├─ Adventure/ (experience images)
├─ assets/ (video, music, pfp)
├─ Backend/ (dhdb.png, nbdb.png)
├─ quest images/ (screenshots for all quests)
├─ UIUX Tasks/ (design task images & videos)
├─ scripts/ (main.js)
└─ styles/ (main.css)
```
